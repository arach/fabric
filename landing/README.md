# Fabric Landing Page

Marketing website for [Fabric](https://github.com/arach/fabric) - ambient compute infrastructure for Claude agents.

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed automatically to [fabric.arach.dev](https://fabric.arach.dev) via GitHub Pages on push to `main`.

## Stack

- React 19
- Vite 6
- TypeScript
- Tailwind CSS (via CDN)
